Mes compétences

Unity

J'ai commencé à utiliser Unity il y a 4 ans maintenant. Et je pense avoir utilisé la plupart des possibilités qu'offre ce moteur et même d'avoir exploré certains sombres recoins (qui a dit les editors tools?). Les bonnes pratiques pour l'optimisation mobile sont également quelques choses que j'affectionne tout particulièrement. Je met l'architecture du code au centre de mon processus de travail. Et lors de l'évolution du code, je n'hésite pas à refactoriser si nécessaire. Mes projets m'ont permis d'apprendre à utiliser Spine et Rewired. Je programme généralement les core systems, tools et le gameplay.

C#, C, Js

Mes connaissances de base en C et en ASM me permettent d'appréhender les languages haut-niveaux sans oublier ce que représente réellement les simplicités de codes. J'applique de nombreux concepts de programmation afin que mon code soit le plus flexible et robuste possible. Notamment en utilisant les bonnes pratiques tel que le DRY( Don't repeat yourself). J'ai également beaucoup utilisé le C# dans d'autre type de projet tel que les site web MVC, Xamarin Forms ou le développement d'application UWP et WPF. J'ai également des connaisances de PHP, Javascript, Html & CSS et certains frameworks web.

Dévelopement Web

Le développement de site web, qu'il soit front ou backend est dans mes capacités. Les projets MVC en C# couplé avec Bootstrap me permettent rapidement de concevoir un site web. J'ai utilisé ces capacités pour notamment développer une plateforme pour gérer les payements d'une épicerie participative bio-éthique-locale afin de gérer les achats des ses coopérateurs-trices.

Autres

Mon background d'ingénieur m'a également permis de réaliser des projets sortants des domaines précédemment évoqués. Que ce soit une machine de production industrielle ou une borne d'arcade. J'ai réalisé divers projets qui m'ont passioné.
